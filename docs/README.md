# Documentation

This folder contains the technical documentation of the AquaSmart smart water control system.

## Documents

- `system_overview.md`  
  High-level system description and architecture overview.

- `control_algorithm.md`  
  Detailed explanation of the PI-based closed-loop control strategy and safety mechanisms.

- `api_reference.md`  
  HTTP API specification for remote monitoring and control.

- `bill_of_materials.md`  
  Bill of Materials (BOM) listing major hardware components used in the prototype.

---

These documents complement the hardware and firmware folders and provide a complete system-level understanding of the project.
